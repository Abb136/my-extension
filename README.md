Bookmark Manager – Chrome Extension
Author: Abbas Dashti

Description
-----------
Bookmark Manager is a simple Chrome extension that allows users to quickly save and manage webpage links directly from the browser popup.

Features
--------
• Save a URL manually.
• Save the currently open tab.
• View saved bookmarks in a list.
• Open bookmarks in a new tab.
• Delete individual bookmarks.
• Delete all saved bookmarks.

How It Works
------------
The extension stores bookmarks locally in the browser using Chrome's local storage system. No data is transmitted to external servers.

Permissions
-----------
tabs
Used to retrieve the URL of the currently active tab when the user clicks "Save Tab".

Installation

For Users
---------
1. Visit the Chrome Web Store listing for Bookmark Manager.
2. Click "Add to Chrome".
3. The extension will be installed automatically and appear in the Chrome toolbar.

For Developers (Testing Only)
-----------------------------
1. Download or clone the extension source code.
2. Open Chrome and navigate to chrome://extensions.
3. Enable "Developer Mode".
4. Click "Load unpacked".
5. Select the extension folder.
Privacy
-------
The extension stores bookmark URLs locally on the user's device. No data is collected, transmitted, or shared with external services.
